高斯消元法介绍

高斯消元法主要用在解线性方程，当前实现的仅仅是有唯一解的一个算法

将线性方程变成一个上三角矩阵的过程

1 2 3 3 4
0 1 2 3 4
0 0 1 2 3
0 0 0 3 3
从这个矩阵也不难看出是一个n x (n+1)类型

实现步骤

  • 实现上三角转换
    主要是进行遍历，遍历的过程主要就是让每行的参数变为0

    第一行遍历剩下的n-1行
    第二行遍历剩下的n-2行
    …
    所有行都遍历

  • 上三角转换后，计算参数
    这里可以利用 n x (n+1)这一特性
